Livret de compétences numérique

Projet fils rouge réalisé lors de ma formation à l'IMIE. C'est une application permettant aux élèves de consulter les résultats de leurs évaluations. L'application possèdent différents rôles: enseignants, les coordinateurs pédagogiques et le directeur.

Technologies utilisées: HTML, CSS, Bootstrap, Javascript, Angular, Java EE, Hibernate, EJB, PostgreSQL, Wildfly, Web Service REST, Authentification.
